# Student Record System (C++)

A menu-driven student record system built using C++ and file handling.  
This project demonstrates how to store and retrieve student data using binary files.

---

## Versions

### 1. Basic Version (`project_v1.cpp`)
- Add student record
- Display stored records
- Simple class-based structure

### 2. Upgraded Version (`project_v2.cpp`)
- Improved file handling using binary read/write
- Better menu structure
- More reliable data storage and retrieval

---

## Concepts Used
- Classes and Objects
- File Handling (`fstream`)
- Binary File Operations (`read`, `write`)
- Menu-driven programming

---

## How It Works
- User selects options from menu:
  - Add → stores student data in file
  - Display → reads and shows stored data
- Data is stored in a file (`data.txt`) using binary format
